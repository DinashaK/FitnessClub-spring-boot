Привет! Меня зовут Диана, я студентка Финансового университета, факультета информационных технологий и анализа больших данных. Этот сайт я разработала с особым вниманием к удобству как для клиентов фитнес-клуба, так и для администраторов и менеджеров. Я сама являюсь клиентом фитнес клуба, и знаю, насколько важно, чтобы сайт был клиентоориентированным, интуитивно понятным и удобным в использовании.

Опыт работы с технологиями

Для разработки сайта я использовала Spring Boot для backend-части, Thymeleaf для динамической генерации HTML и Bootstrap для стилизации интерфейса. Для работы с данными применяла JPA, что обеспечило эффективное взаимодействие с базой данных.
